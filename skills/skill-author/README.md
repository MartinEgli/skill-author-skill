# Skill Author

Use `skill-author` when creating, refining, validating, releasing, or cataloging
agent skills in AgentSkills.

Typical work:

- create a new skill repo from `single-skill-template`
- define triggers, boundaries, modes, handoffs, and output contracts
- add evidence and traceability behavior
- align `agents/openai.yaml`
- document target-agent compatibility for Codex, Claude, Cursor, Windsurf,
  Cline, or another host
- validate, package, commit, push, and update the AgentSkills submodule pointer
- update AgentSkills README/AGENTS so the catalog remains current
